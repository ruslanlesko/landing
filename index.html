Personal Projects

All my pet projects can be found on my GitHub page. They are my place to try and learn new technologies, so all of them are not production-level. However, my pet projects are suitable to be used by other people for non-production needs. Here is the list of the most interesting ones.

Digestwave

News aggregator which doesn't spy on you and allows adding new sources by committing into the source code. Available at digestwave.com

BrightonUM

BrightonUM is simple user management and authentication service. I have created it for my private project: Lifecapsule. BrightonUM can be used in any small project which needs a user management system.

PalermoPG

PalermoPG is a server for self-hosted picture gallery. It is a back-end for my private gallery service: Lifecapsule. PalermoPG is powered by Java with Vert.x library.
